✈️ 飞鸟特价 - 智能特价机票发现平台

📝 项目简介

“飞鸟特价” 是一个专注于发现全网隐藏特价机票、智能解读航司规则（防坑）的前端交互演示平台。

本项目为纯前端零依赖的单页面应用 (SPA)。完全使用原生 HTML、CSS 和 Vanilla JavaScript 构建，自主实现了前端路由、组件渲染、Canvas 数据可视化以及基于 LocalStorage 的状态管理。

(注：本项目仅作为前端技术能力展示与测评演示工具，数据均为前端 Mock 生成，不包含真实支付接口。)

✨ 核心功能亮点

1. 🔍 智能检索与多维筛选

支持出发地/目的地拼音与汉字模糊匹配（如输入 bj 匹配北京）。

提供完善的特价机票池，支持按“价格”、“起飞时间”、“性价比”排序。

支持无缝筛选航司类型（全服务/廉航）、是否直飞以及动态价格区间拉杆。

2. 🤖 AI 规则大白话解读 (核心防坑)

针对廉价航空（LCC）的“行李额度”、“餐食”、“苛刻退改签”等隐藏陷阱，提供醒目的红色高亮警告。

动态生成阶梯退改手续费表格，直观对比当前机票与同航线的“规则宽松度 (Flex Score)”。

3. 📉 数据可视化与比价

Canvas 价格走势图：纯原生 JS 结合 Canvas API 绘制的近 30 天历史价格波动图表。

多渠道 OTA 价格模拟比对，辅助决策。

4. 🔔 降价监控订阅系统

模拟真实的降价订阅逻辑，支持设置心理预期价位、出行时间灵活度。

使用 LocalStorage 实现数据的本地持久化，支持订阅记录的保存、状态切换、备份与恢复功能。

5. 📱 极致的响应式体验

采用 CSS Variables 结合媒体查询 (Media Queries)，完美适配桌面端、平板与移动端设备。

移动端自动切换为底部原生 App 风格的导航栏 (Bottom Nav)，操作顺滑。

🛠️ 技术实现细节

本项目最大的特色在于**“零外部依赖”**（Zero Dependencies）：

路由控制：基于 window.onhashchange 监听自主实现前端 Hash 路由，实现页面无刷新切换 (#/, #/results, #/details/:id, #/alerts)。

状态管理：全量使用原生 JS 变量维护应用状态，使用 JSON 序列化配合 LocalStorage 实现数据本地存储。

UI & 动效：全部 CSS 手写，包含自定义变量主题、流光扫过动画 (@keyframes shine)、呼吸灯警告框等。
